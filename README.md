<h1 align="center">
  Ignite Timer
</h1>


A React.JS application to countdown desired task time. Fully frontend with React and Vite.

## :pushpin: Technologies
- Typescript
- React.JS
- Vite
- NPM

## :pushpin: Libraries
- styled-components
- date-fns
- immer
- phosphor-react
- react-hook-form
- hookform/resolvers
- zod

## :computer: How to run
```
npm install

npm run dev

// It will run on localhost:5173
```
